Script intended for Windows OS. Requires an active antivirus or EDR solution.

This script is created to test antivirus and EDR-functions by creating, writing and reading a known malware string with a Python-script.

An EICAR-string will be used in the code that an antivirus should detect as malware. The string is completely harmless to the computer.

There is NO malware in the script. It is created for educational purposes.


How to use:
- Open a terminal (CMD or Powershell)
- Locate the script in the correct directory
- Run the script by typing in: python script.py
